## What is lazyaz?

`lazyaz` is to `az` what `lazygit` is to `git`: a fast, fully-keyboard terminal explorer that
turns common Azure browse-and-inspect tasks into a few keystrokes. No more clicking through the
Azure Portal to peek at a dead-letter queue or download a blob.

It covers three Azure resource types — the ones developers actually touch day-to-day:

- **Blob Storage** — browse containers, navigate virtual folders, preview files with syntax highlighting, multi-select and download
- **Service Bus** — peek active and dead-letter queues, requeue DLQ messages, inspect topics and subscriptions
- **Key Vault** — list vaults, browse secrets and their versions, yank values to clipboard

## Who is it for?

Developers and platform engineers who work with Azure regularly and want a fast, keyboard-driven
alternative to the portal. If you spend time browsing storage accounts, tailing dead-letter queues,
or pulling secrets during debugging sessions, this is for you.
