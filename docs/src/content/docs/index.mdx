## What is lazyaz?

`lazyaz` is to `az` what `lazygit` is to `git`: a fast, fully-keyboard terminal explorer that
turns common Azure browse-and-inspect tasks into a few keystrokes. No more clicking through the
Azure Portal to peek at a dead-letter queue or download a blob.

## Who is it for?

Developers and platform engineers who work with Azure regularly and want a fast, keyboard-driven
alternative to the portal. If you spend time browsing storage accounts, tailing dead-letter queues,
or pulling secrets during debugging sessions, this is for you.

## Feature highlights

<CardGrid>
	<Card title="Blob Storage" icon="document">
		Browse containers, navigate virtual folders, preview files with syntax highlighting, multi-select and download in bulk. Server-side prefix search on containers of any size.
	</Card>
	<Card title="Service Bus" icon="warning">
		Peek active and dead-letter queues, requeue or move DLQ messages, delete duplicates, and inspect topics and subscriptions — without writing CLI one-liners.
	</Card>
	<Card title="Key Vault" icon="setting">
		Browse secrets and their version history. Yank a secret value to your clipboard with a single keystroke, or bulk-export marked secrets as JSON.
	</Card>
	<Card title="Tabbed multi-service UI" icon="open-book">
		Run Blob, Service Bus, and Key Vault side by side in tabs. Each tab has its own subscription. Switch with `Alt+1`–`9` or the command palette (`Ctrl+P`).
	</Card>
	<Card title="Action menu & command palette" icon="list-format">
		Press `a` for context-aware actions on the current selection, or `Ctrl+P` to search all available commands. Every operation is a few keystrokes away.
	</Card>
	<Card title="Customizable themes & keybindings" icon="puzzle">
		Ships with 20+ Base16 color schemes. Keybindings are fully remappable via JSON — vim-style by default.
	</Card>
</CardGrid>
