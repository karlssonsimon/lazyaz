Browse Azure Blob Storage, Service Bus, and Key Vault from your terminal.
Navigate storage accounts, peek dead-letter queues, yank secrets — all without
leaving the keyboard. Runs anywhere `az login` works.

## Prerequisites

- **Go 1.22+** (for `go install`)
- **Azure CLI** (`az`) — [install guide](https://learn.microsoft.com/en-us/cli/azure/install-azure-cli)
- An active Azure subscription with resources to browse

## Installation

<Tabs>
<TabItem label="go install">
```bash
go install github.com/karlssonsimon/lazyaz/cmd/lazyaz@latest
```
</TabItem>
<TabItem label="Build from source">
```bash
git clone https://github.com/karlssonsimon/lazyaz.git
cd lazyaz
go build -o lazyaz ./cmd/lazyaz
```
</TabItem>
</Tabs>

## Log in to Azure

Make sure you're authenticated:

```bash
az login
```

If you work with multiple tenants, don't worry — lazyaz supports [switching tenants](/lazyaz/multi-tenant/) from within the app without re-opening a browser.

## Launch

```bash
lazyaz
```

## Your first 60 seconds

lazyaz ships with two keymaps: **vim-style** (default) and **standard**. Switch by setting `keymap` in `~/.config/lazyaz/config.json`:

```json
{ "keymap": "standard" }
```

<Tabs syncKey="keymap">
<TabItem label="Vim (default)">
<Steps>

1. **Pick a subscription** — the subscription picker opens automatically. Type to fuzzy-filter, press `Enter` to select.

2. **Browse resources** — storage accounts, namespaces, or vaults appear in the left pane. Press `Enter` or `l` to drill in.

3. **Navigate** — `j`/`k` to move the cursor, `Tab` to switch panes, `h` to go back. It works like you'd expect.

4. **Use the action menu** — press `a` for context-aware actions: download blobs, requeue dead-letter messages, yank secrets.

5. **Open the command palette** — press `Ctrl+P` to search all commands: open new tabs, switch themes, change subscriptions, switch tenants.

</Steps>
</TabItem>
<TabItem label="Standard">
<Steps>

1. **Pick a subscription** — the subscription picker opens automatically. Type to fuzzy-filter, press `Enter` to select.

2. **Browse resources** — storage accounts, namespaces, or vaults appear in the left pane. Press `Enter` or `Right` to drill in.

3. **Navigate** — arrow keys to move, `Tab` to switch panes, `Left` to go back.

4. **Use the action menu** — press `a` for context-aware actions: download blobs, requeue dead-letter messages, yank secrets.

5. **Open the command palette** — press `Ctrl+P` to search all commands: open new tabs, switch themes, change subscriptions, switch tenants.

</Steps>

## Pre-configured tabs

Skip the subscription picker by configuring startup tabs in `~/.config/lazyaz/config.json`:

```json
{
  "tabs": [
    { "kind": "blob", "subscription": "your-subscription-id" },
    { "kind": "servicebus", "subscription": "your-subscription-id" },
    { "kind": "keyvault" }
  ]
}
```

lazyaz opens directly into these tabs on launch. See [Configuration](/lazyaz/configuration/overview/) for all options.
