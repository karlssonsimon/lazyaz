The Service Bus view lets you browse namespaces, queues, and topics. You can
peek messages from active and dead-letter queues, requeue or move DLQ messages,
and manage topic subscriptions.

## Pane layout

The pane structure adapts as you navigate deeper:

**Namespaces** → **Entities** → **Subscriptions** (topics only) → **Queue Type** → **Messages** → **Message Preview**

Not all panes are visible at once — the view shows a parent, focused, and
child pane in a miller-columns layout, and adjusts as you navigate.

## Browsing entities

Entities are queues and topics within a namespace. Press `Enter` on a namespace
to load its entities.

For **queues**, pressing `Enter` opens the queue type picker where you choose
between the active queue and the dead-letter queue.

For **topics**, pressing `Enter` shows the topic's subscriptions. Select a
subscription to reach the queue type picker.

### Sorting and filtering entities

Press `a` to open the action menu, then select "Sort entities" to sort by:

- Default order
- Name (ascending / descending)
- Active message count (ascending / descending)
- Dead-letter count (ascending / descending)

Use `f` to toggle the DLQ filter, which hides entities with zero dead-letter messages.

## Peeking messages

Select "Active" or "DLQ" in the queue type picker, and messages are peeked
automatically. Peeking is read-only — it does not consume or lock messages.

Press `Enter` on a message to open the detail preview with the full message
body. You can select text in the preview with the mouse to copy it.

| Key | Action |
|---|---|
| `[` | Switch to active queue |
| `]` | Switch to dead-letter queue |
| `Enter` | View message detail |
| `h` / `Esc` | Close message preview |

## Dead-letter queue operations

<Aside type="caution">
DLQ operations modify messages. Requeue and move are non-destructive (messages
are copied to the target, then completed from DLQ), but complete permanently
removes messages.
</Aside>

When viewing the dead-letter queue, you can operate on messages through the
action menu (`a`):

**Single / marked messages:**
- **Requeue** — move back to the active queue
- **Move to...** — transfer to a different queue or subscription
- **Complete** — permanently remove from the DLQ

**Bulk operations:**
- **Requeue all** (`R`) — requeue every message in the DLQ
- **Move all to...** — move all DLQ messages to another target
- **Delete duplicates** (`D`) — remove messages with duplicate message IDs

DLQ operations use receive-with-lock under the hood. The stream manager (`F`)
shows progress for long-running operations, and you can cancel them mid-flight.

## Selecting messages

Mark individual messages with `Space`, or use visual-line mode (`v`) to select
a range. Marked messages can be operated on in bulk through the action menu.

## Inspect panel

Press `K` to toggle the inspect strip, which shows details for the selected
namespace, entity, or message — including message counts, metadata, and
properties.
