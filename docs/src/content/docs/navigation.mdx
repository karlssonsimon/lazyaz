## Pane layout

lazyaz uses a **miller-columns** layout вЂ” parent, focused, and child panes
arranged left to right. As you drill into resources, the focused pane shifts
right and the previous pane becomes the parent column on the left.

The **Notifications history** (`N`) overlay keeps the last 1 000 events.
Transient toasts auto-dismiss after ~1.5 s (or 5 s for errors). The
**Activity overlay** (`F`) shows in-flight cache fetches across all tabs
and lets you cancel them mid-flight вЂ” note that some long-running
operations (Service Bus requeue/move) are not yet routed through it.

## Action menu

Press `a` to open a context-aware action menu for the current pane. The menu
shows only actions relevant to your current selection вЂ” download, yank, sort,
requeue, and more depending on the resource type. Type to filter, `Enter` to
execute.

## Command palette

Press `Ctrl+P` to open the command palette. It lists every available
command grouped by section вЂ” new tabs (Blob / Service Bus / Key Vault /
Dashboard / Azurite preset / Blob-by-connection-string), open tabs (`Go to
Tab N`), tab management (close / next / previous), the **Azure Login /
Switch Tenant** flow, theme picker, notifications history, activity
overlay, help, and quit. Type to fuzzy-filter, `Enter` to execute.

Note: switching subscription is not in the palette вЂ” it's the top-level
shortcut `S` (or `Ctrl+S` in standard).

## Jump list

lazyaz keeps a vim-style jump list that tracks where you've been across
tabs. Drill-ins (into a container, namespace, vault) plus tab switches and
new-tab opens are recorded.

| Key | Action |
|---|---|
| `Ctrl+O` | Jump back to the previous position |
| `Ctrl+I` / `Ctrl+N` / `Ctrl+]` | Jump forward |

Jumps cross tab boundaries вЂ” pressing `Ctrl+O` can return you to a different
tab you had open. The list holds up to 100 entries and is deduplicated so
rapid back-and-forth doesn't bloat it.

Some terminal + multiplexer combinations can't distinguish `Ctrl+I` from `Tab`,
which is why `Ctrl+N` and `Ctrl+]` are also bound to jump forward.
